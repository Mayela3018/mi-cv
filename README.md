CV profesional desarrollado con **HTML5 y CSS3 puro**, sin frameworks ni dependencias externas. Cuenta con un diseño de dos columnas, animaciones suaves y diseño completamente responsivo para móvil y escritorio.

## 🚀 Cómo ejecutar localmente

### Opción 1 — Live Server en VS Code (recomendado)
1. Abre la carpeta en **VS Code**
2. Instala la extensión **Live Server**
3. Clic derecho en `index.html` → **"Open with Live Server"**
4. Se abre en `http://127.0.0.1:5500`

### Opción 2 — Abrir directo en el navegador
1. Ve a la carpeta del proyecto
2. Doble clic en `index.html`
3. ¡Se abre directamente en tu navegador!

## ☁️ Despliegue en Render

### Configuración del Static Site

| Campo | Valor |
|-------|-------|
| Tipo | Static Site |
| Branch | `main` |
| Publish Directory | `.` |
| Build Command | *(vacío)* |

### Pasos para deployar
1. Sube el proyecto a **GitHub**
2. Ve a [render.com](https://render.com) → **New +** → **Static Site**
3. Conecta tu repositorio
4. Configura el **Publish Directory** como `.`
5. Clic en **"Create Static Site"**

Render desplegará automáticamente cada vez que hagas `git push` 🎉
